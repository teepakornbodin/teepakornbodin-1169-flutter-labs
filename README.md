📚 Flutter Labs: teepakornbodin-1169-flutter-labs

Repository นี้เป็นแหล่งรวบรวมโปรเจกต์และแบบฝึกหัด (Labs) ที่เกี่ยวข้องกับการเรียนรู้และพัฒนาแอปพลิเคชันด้วย Flutter Framework โดยมีวัตถุประสงค์เพื่อฝึกฝนทักษะในการสร้างส่วนติดต่อผู้ใช้ (UI), การจัดการสถานะ (State Management), การเชื่อมต่อ API, และฟีเจอร์อื่นๆ ที่สำคัญในการพัฒนาแอปพลิเคชันข้ามแพลตฟอร์ม (Cross-Platform)
📌 เนื้อหาหลัก
Repository นี้ประกอบด้วยโฟลเดอร์สำหรับแต่ละแล็บหรือโปรเจกต์ย่อย โดยทั่วไปแล้วจะครอบคลุมหัวข้อต่างๆ ดังนี้

- พื้นฐานของ Widget: การใช้งาน Stateless และ Stateful Widgets, Layouts (Row, Column, Container)
- การนำทาง (Navigation): การเปลี่ยนหน้าจอ (Routes), การส่งข้อมูลระหว่างหน้า
- การจัดการสถานะ (State Management): การเรียนรู้แนวคิดและเครื่องมือต่างๆ (เช่น Provider, Bloc, Riverpod) ในการจัดการข้อมูล
- การจัดการ Input: การใช้งาน Forms, TextField, Validation
- การเชื่อมต่อข้อมูล: การใช้งาน http package เพื่อเรียกข้อมูลจาก REST API
- การจัดเก็บข้อมูลภายในเครื่อง: การใช้งาน Shared Preferences หรือฐานข้อมูล
